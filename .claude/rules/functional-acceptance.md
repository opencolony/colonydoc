# 功能验收

### 完成功能或修复后：

- 运行 `npm run typecheck` 验证无类型错误
- **分析变更是否需要更新 README 和 README.zh**，如需要则进行更新

### 以下主要功能修改后，必须使用 chrome-devtools MCP 进行功能测试：

- **编辑器**：编辑、预览模式切换、Markdown 渲染、内容保存
- **文件树**：文件/目录的增删改查、拖拽排序、展开折叠
- **搜索**：全文搜索、搜索结果显示、跳转定位
- **多标签页**：Tab 打开/关闭/切换、脏状态提示、自动保存
- **图表与公式**：Mermaid 图表渲染、LaTeX 公式渲染
- **移动端适配**：侧边栏、工具栏、对话框等移动端交互

> **注意**：以上功能列表为白名单。对于未列出的功能模块（如纯后端 API、配置加载逻辑、工具函数、类型定义、构建配置等），**未经允许禁止使用 chrome-devtools MCP 进行测试**。

通过浏览器自动化验证功能在移动端和桌面端正常工作。使用前先确定 **5787** 端口是否已启动，如果启动则直接测试，否则先使用 `pnpm dev` 启动
